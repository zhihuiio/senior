# Contributing to Senior

Thanks for your interest in contributing to Senior.

## Development Setup

Prerequisites:

- Node.js 20+
- npm 10+
- Desktop GUI environment (Electron)

Install and run:

```bash
npm install
npm run dev
```

Build and test:

```bash
npm run build
npm run test:freeform-agent
```

## Pull Request Guidelines

- Keep changes focused and atomic.
- Include tests for behavior changes when possible.
- Update docs when public behavior, workflows, or commands change.
- Ensure local build and tests pass before opening a PR.

## Commit Guidance

- Use clear commit messages that explain intent and scope.
- Avoid mixing refactors with behavior changes in the same commit.

## Reporting Bugs

- Open an issue with reproduction steps, expected behavior, and actual behavior.
- Include environment details (OS, Node/npm versions, and app version).

## Security

For security issues, do not open a public issue first. See `SECURITY.md`.
